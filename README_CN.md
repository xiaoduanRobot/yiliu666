## 🔄 自我修复功能与备用修复方案

小端界面干净，只有设置模型，2种模式切换，普通模式适合本地模型或按TOKEN计算，复杂模式适合按次数算

隐藏功能，点击聊天框上方智能AI助手，开启自动进化模式，新手谨慎使用

（大模型修改删除文件操作都会自动触发该文件备份，保留3天自动清理，若配置改坏无法对话，小端检测5分钟大模型无回复，会自动恢复，过5分钟重启即可)

（备用方案：配置改坏后在C/users/xiaoduan文件夹中的记忆.txt，jineng目录复制出来，删除xiaoduan文件夹重启小端即可恢复，再替换回 记忆.txt,jineng目录）

## 📚 jineng技能库

### 什么是jineng

`jineng`是小端在魔搭社区的技能库文件夹，包含各种专业人士已跑通的自动化技能。交给作者严格审，记录纯中文MD，仅记录执行流程，不含任何PY执行文件，安全可靠！

#### 四重审核
为什么会更安全可靠：小端的技能，是 `.md` 文本文件，描述"怎么做"，而不是"直接运行skills里写好的py代码"。如果大模型运行别人的.PY他不会看，只会执行，忽略漏洞，但大模型自己写.PY则不会。

1. **架构隔离**：技能文件只有纯中文的流程介绍，无法执行命令，流程国人一看就懂
2. **模型自审**：大模型根据文档生成技能，写代码验证时，内置的安全对齐会过滤掉"不应该生成"的内容。进行第一次审核
3. **作者审核**：每一个放在魔搭社区的共享技能，我亲自检查，进行第二次审核
4. **平台审核**：魔搭社区每次提交文档会严格审查，有时两三天才通过，进行第三次审核
