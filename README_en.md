# 🤖 Xiaoduan AI Guidelines

(*For commercial use, please credit the author: yiliu. Respect the author's months of hard work.*)

Xiaoduan is 100% open source and runs locally. The author has not built any servers. The only official download source is ModelScope (魔搭社区).

To upgrade to a new version or fix a failed modification, simply uninstall and reinstall. Your permanent memory and existing configurations will not be lost.

Default 4 free models available. You can configure APIs using natural language. If the free models are unstable, try switching or add your own API manually.

Connect to QQ or Feishu first. After that, you can send voice messages directly. If you have any questions, just ask Xiaoduan - everything can be solved.

## 🔄 Self-Evolution Feature

Xiaoduan has a clean interface with only model settings and 2 mode switches. Normal mode is suitable for local models or token-based billing. Complex mode is for per-request billing.
Hidden feature: Click "AI Assistant" at the top of the chat box to enable auto-evolution mode. Beginners use with caution.
(Any file modification or deletion triggers automatic backup, keeping 3 days of history. If you break something, Xiaoduan can recover itself - no need to worry!)

### What Can Xiaoduan Do

Xiaoduan has **self-evolution** capabilities:

| Capability | Description |
|------------|-------------|
| **Learn New Skills** | Automatically browse the web to learn good automation techniques |
| **Record Experience** | Write successful workflows as MD documents for permanent storage |
| **Cloud Sync** | Contact the author to upload to ModelScope for all Xiaoduan instances to share |
| **Learn by Doing** | Solve problems as they arise, record solutions |
| **Security** | Only reference MD document workflows, Xiaoduan writes its own code - safe and reliable |

### Evolution Workflow

```
New Task → Analyze Requirements → Find Methods → Execute & Verify → Document → Save to Skills Directory → Continue Verification → Continue Documentation → Success → Clean Up Errors → Keep Complete Clean Skills
```
