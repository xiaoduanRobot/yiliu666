# Agent-Browser 浏览器自动化代理

## 技能简介
agent-browser是一款浏览器自动化工具，专门用于网页表单自动填写、网页交互自动化和网站数据抓取。该工具通过命令行界面提供导航、快照分析、元素交互、信息获取等功能。

## 核心功能
- 导航(Navigate) - 打开网页
- 快照(Snapshot) - 获取页面元素列表
- 元素交互 - 点击、输入、悬停、滚动等
- 信息获取 - 抓取网页数据
- 表单填写 - 自动填写网页表单
- 数据采集 - 批量抓取网页数据

## 使用场景
- 自动化测试
- 网页爬虫
- 表单自动填写
- 数据采集
- RPA机器人
- 工作流自动化

## 工具命令
- navigate - 导航到URL
- snapshot - 获取页面快照
- click - 点击元素
- type - 输入文本
- hover - 悬停
- scroll - 滚动页面
- evaluate - 执行JS代码
- wait - 等待元素
